Product Page

The purpose of this project was to utlize sketching, ideation, and gestalt principles to create the product page for a tea company client. I learned a lot about the importance of proximity, similarity, closure, symmetry, hierarchy, contrast, and alignment while working on this project. I generated the images using OpenAI's DALL-E platform in order to create consistent and aesthetically pleasing imagery.
